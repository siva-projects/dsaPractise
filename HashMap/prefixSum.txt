- Prefix Sum is used to calculate sub arrays sum .

Template 
    - build a prefix Sum array where prefix[right] - prefix[left-1] = sum of indexes from ( left, right);
    - if left == 0 then prefix[right] will be the sum.

- when dealing with questions such as " 560. Subarray Sum Equals K ", we usually put prefix[0] = 0 or freqMap.out(0,1) to include the single element ( index right )
- Below is the example 

class Solution {
    public int subarraySum(int[] nums, int k) {
        Map<Integer, Integer> freqMap = new HashMap<>();
        freqMap.put(0, 1);

        int sum = 0;
        int ans = 0;
        for(int num : nums){
            sum += num;
            if(freqMap.containsKey(sum - k)){
                ans += freqMap.get(sum - k);
            }
            freqMap.put(sum, freqMap.getOrDefault(sum, 0) + 1);
        }
        return ans;
    }
}
